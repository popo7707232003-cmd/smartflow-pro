# ⚡ SmartFlow Pro — 10 維聰明錢監控系統

## 功能
- **即時價格** — AI 搜尋真實幣價（每 25 秒更新）
- **10 維訊號引擎** — 趨勢、動量、量能、聰明錢、資金費率、恐懼貪婪、清算、多空比、穩定幣、交易所存量
- **聰明錢追蹤** — 鯨魚動態、交易所資金流向、大戶吸籌/出貨
- **消息面監控** — AI 即時搜尋翻譯加密貨幣新聞
- **即時績效** — 勝率、利潤因子、交易紀錄、PnL

## 🚀 部署到 Vercel（完全免費）

### 步驟 1：下載並解壓此專案

### 步驟 2：上傳到 GitHub
1. 到 github.com 登入（或註冊免費帳號）
2. 點右上角 + → New repository
3. 名稱填 smartflow-pro，點 Create
4. 點 "uploading an existing file"
5. 把解壓後的所有檔案拖進去 → Commit

### 步驟 3：部署到 Vercel
1. 到 vercel.com 用 GitHub 登入
2. 點 "Add New..." → Project
3. 選 smartflow-pro → Import
4. Framework Preset 選 Vite
5. 點 Deploy
6. 約 1 分鐘後拿到免費網址！

### 步驟 4：分享
把網址傳給任何人，打開即用！
